# ART-IN-MOTION
Dance & Stage Academy website project

Art in Motion is a responsive website for a fictional dance and stage academy.  
The project presents different dance styles and allows users to explore classes, view a gallery and apply for a course.

---

## Features

### Homepage
- Introduction to the academy
- Overview of dance styles (Ballet, Modern Dance, Folk Dance, Stage Skills)
- Clean and minimal layout

### Subpages
- Individual pages for each dance style
- Consistent layout and typography
- Image and text combination for each style

### Gallery
- Image grid layout
- Clickable images with preview (Lightbox)
- Smooth hover effects

### Contact Form
- User application form
- Multiple input types:
  - text
  - email
  - number
  - select
  - radio buttons
  - textarea
  - checkbox
- JavaScript validation
- Error messages displayed under each field

### JSON + AJAX
- Data stored in `classes.json`
- Nested structure (academy → programs → details → schedule)
- Loaded using JavaScript (fetch)
- Dynamically displayed in a table on the contact page

---

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid)
